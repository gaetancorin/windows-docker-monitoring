-------------- HELP DOCKER COMPOSE ---------------
build docker compose containers:
docker-compose -p monitoring_streaming up --build -d

delete all containers and all volumes
docker-compose -p monitoring_streaming down -v

delete all containers
docker-compose -p monitoring_streaming down

----------------------------------
FIRST RUN TUTORIAL
1- Go to Docker Desktop → Settings → General → check “Expose daemon on tcp://localhost:2375 without TLS”
2- Restart Docker Desktop

(Tool 'cAdvisor' does not work properly because of windows WSL, so docker-exporter is used instead)
